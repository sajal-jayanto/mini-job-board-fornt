## Default User

```bash
  name: "admin",
  email: "admin@admin.com",
  password: "admin@pass"
```

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# build
$ npm run build

# development
$ npm run dev

# production mode
$ npm run start

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
